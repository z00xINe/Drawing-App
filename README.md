# Drawing App

A simple and interactive Drawing Application that allows users to draw freely on a canvas using different tools and colors.

## Overview
This project is a lightweight drawing tool built to demonstrate how canvas-based rendering works in web applications. It allows users to create sketches, experiment with colors, and interact with graphical elements in real-time.

## Features
- Freehand drawing on canvas
- Color selection tool
- Adjustable brush size
- Clear canvas functionality
- Smooth user interaction with mouse events
- Real-time rendering

## Technologies Used
- HTML5 Canvas
- JavaScript
- CSS

## How It Works
The application uses the HTML5 Canvas API to capture mouse movements and render drawings dynamically on the screen. Users can draw by clicking and dragging the mouse across the canvas.
